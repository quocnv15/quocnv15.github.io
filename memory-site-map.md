## Memory Navigation Map

Đây là bản đồ đọc nhanh cho hệ tài liệu memory theo thứ tự ưu tiên.

### 1) Start Here

- [Memory Hub Summary](/memory/): tóm tắt toàn bộ trong 5-10 phút.
- [Dashboard](/dashboard/): ưu tiên hành động tuần hiện tại.
- [Portfolio](/portfolio/): catalog app và hướng dài hạn.

### 2) Portfolio Analysis Tracks

- Business snapshot: active apps, ROI, status theo app.
- Category lens: utility camera vs wallpaper vs game.
- Monetization lens: IAP impulse vs subscription mix.
- Tech debt lens: issues hệ thống cần fix cấp nền.

### 3) Weekly Execution Tracks

- Scale: app đang thắng cần tăng tốc có kiểm soát.
- Optimize: app ổn định cần nâng biên lợi nhuận.
- Watch: app mới chưa đủ dữ liệu, theo dõi tín hiệu.

### 4) Suggested Reading Flow

1. `/memory/`
2. `/memory/portfolio-overview/`
3. `/memory/quality-gaps/`
4. `/memory/showcase/`
5. `/memory/mentors/`
6. quay lại `/dashboard/` để chốt action tuần.
